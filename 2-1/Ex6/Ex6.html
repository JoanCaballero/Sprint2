Array Loops

Exercici 6.1

Declaració de la funció:

        
          const noms = ['Anna', 'Bernat', 'Clara']
          const imprimirNoms = (noms) => {
            noms.forEach(nom => console.log(nom));
          }

Codi d'execució:

        
          imprimirNoms(noms)

Exercici 6.2

Declaració de la funció:

        
          const imprimirNomsForOf = (noms) => {
            for (let nom of noms) {
              console.log(nom);
            }
          }

Codi d'execució:

        
          imprimirNomsForOf(noms)

Exercici 6.3

Declaració de la funció:

        
          const numeros = [1, 2, 3, 4, 5, 6]
          let numerosParells = numeros.filter(num => num % 2 === 0)

Codi d'execució:

        
          console.log(numerosParells)

Exercici 6.4

Declaració de la funció:

        
          const obj = {nom: 'Ona', edat: 25, ciutat: 'Barcelona'}
          const imprimirObj = (obj) => {
            for (let key in obj) {
              console.log(`${key}: ${obj[key]}`);
            }
          }

Codi d'execució:

        
          imprimirObj(obj)
